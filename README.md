# ios_phonecallwidget

Widget implementation:

`pod 'PhoneCallModule', :git => 'git@gitlab.vladimir.ibuildapp.com:ios/phonecallwidget.git', :branch => 'master'`
 
Data parameters:
title - button name
#phone - phone number
Example:
`
{
    "widgetid": "36556535",
    "type": "taptocall",
    "data": {
        "title": "Button",
        "#phone": "+1(345)435-6234"
    }
}`
